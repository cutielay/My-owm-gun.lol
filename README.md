# Open Source guns.lol Style Profile

A modern **guns.lol inspired profile page** with a dark neon aesthetic, glowing UI, animated stars, Discord card, social icons, and smooth animations.

Built for anyone who wants a stylish personal profile page and wants to customize it easily.

## ✨ Features

- 🌌 Animated star background
- 💜 Purple neon aesthetic
- ⚡ Smooth hover animations
- 🎮 Social media icons section
- 💬 Discord server card
- 📱 Mobile responsive
- ✨ Mouse glow effect
- 🔧 Easy to edit and customize

## 📂 Files

```txt
index.html  → Main page
style.css   → Styling & animations
script.js   → Mouse glow & interactions
README.md   → Project info
```

## 🚀 Installation

### 1. Download the files
Download or clone this repository.

```bash
git clone https://github.com/YOUR-USERNAME/YOUR-REPO.git
```

### 2. Open locally
Simply open:

```txt
index.html
```

in your browser.

Or use VSCode + Live Server.

## 🌍 Deploy to GitHub Pages

1. Upload files to your GitHub repo
2. Go to **Settings**
3. Click **Pages**
4. Under **Source**, choose:

```txt
Deploy from branch
```

5. Select:

```txt
main / root
```

6. Save changes

Your website will be live at:

```txt
https://yourusername.github.io/repository-name/
```

## 🎨 Customization

### Change Username

Inside `index.html`:

```html
<h1>YOUR NAME</h1>
```

### Change Bio

```html
<p class="bio">Your bio here</p>
```

### Change Social Links

Replace `#` with your links:

```html
<a href="https://discord.gg/yourinvite">💬</a>
```

### Change Colors

Inside `style.css` search for:

```css
#8f7cff
```

Replace it with your favorite color.
